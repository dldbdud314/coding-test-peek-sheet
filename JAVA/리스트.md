### Sorting

````java
import java.util.*;

// 배열 정렬
Arrays.sort(arr);
Arrays.sort(arr,Collections.reverseOrder());
Arrays.sort(arr, new Comparator<Integer>() { // 직접 구현
    @Override
    public int compare(Integer i1, Integer i2) {
        return i2 - i1;
    }
});
Arrays.sort(arr, (i1, i2) -> i2 - i1); // 람다 활용

// 리스트 정렬
Collections.sort(list);
````
