### 주요 문법들

````java
HashMap<Integer, String> map = new HashMap<>();

int size = map.size();

// 추가
map.put(1, "StringVal");

// 삭제
map.remove(1);

map.containsKey(1);
map.containsValue("StringVal");

// value / key 모음
map.values();
map.keys();

// 순회하기
map.forEach((key, value) -> {
    System.out.println("KEY: " +  key + " / VALUE: " + value);
});
````
